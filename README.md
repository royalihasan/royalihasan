**[Texurix](https://github.com/royalihasan/texurix)** — A telecom-grade SMS gateway with real-time AI spam detection. Every outbound message passes through a Hybrid CNN model trained on 55,000 messages before reaching the carrier. Built with Java Spring Boot, Python FastAPI, Apache Kafka, PostgreSQL, and Next.js. Achieves greater than 95% spam detection accuracy with less than 5% false positive rate.
